🌐 How cross-network streaming works:

This app uses WebRTC (via PeerJS) — the same technology that powers video calls (Zoom, Google Meet). When the host plays music, the audio stream is sent directly peer-to-peer to each listener's device across the internet. No server needed to relay the audio. PeerJS provides a free public signaling server to help devices find each other.

Requirements: Both host and listeners need a modern browser (Chrome, Firefox, Edge, Safari 15+). Works on phones, laptops, and desktops — anywhere in the world with an internet connection.
